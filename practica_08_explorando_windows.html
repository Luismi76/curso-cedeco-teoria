Practica 08: Explorando Windows 10 en Maquina Virtual

Modulo: MF0219_2 - Instalacion y Configuracion de Sistemas Operativos
Unidad Formativa: UF0853 - Explotacion de las funcionalidades del sistema microinformatico
Bloque: Funcionamiento de Sistemas Operativos
Duracion estimada: 5 horas (1 sesion)
Modalidad: Trabajo individual
Herramientas: Oracle VirtualBox + VM con Windows 10 instalado (Practica 06)
Nota: Esta practica requiere tener completada la Practica 06 (Instalacion de Windows). Trabajaras sobre la maquina virtual de Windows 10 que ya tienes creada.

Objetivos de la practica

  • Conocer la estructura de directorios de Windows y gestionar archivos con el Explorador.
  • Utilizar el Panel de Control y la aplicacion de Configuracion para ajustar el sistema.
  • Monitorizar procesos y rendimiento con el Administrador de tareas.
  • Ejecutar comandos basicos en CMD y PowerShell.
  • Gestionar cuentas de usuario locales.
  • Utilizar las herramientas de mantenimiento del sistema.

Requisitos previos

Antes de comenzar, comprueba que cumples todos los requisitos:

Requisito OK
Practica 06 (Instalacion de Windows) completada
Maquina virtual con Windows 10 funcionando correctamente
Guest Additions de VirtualBox instaladas

Parte 1 - Explorador de archivos y estructura de directorios (30 min)

Procedimiento

  1. Inicia la maquina virtual de Windows 10 y abre el Explorador de archivos (tecla Windows + E).
  2. Navega hasta la unidad C:\ y observa las carpetas principales.
  3. Identifica y documenta las siguientes carpetas del sistema.

Carpetas del sistema

Carpeta Que contiene Puedes modificar su contenido?
C:\Windows
C:\Program Files
C:\Program Files (x86)
C:\Users
C:\Users\[tu usuario]\Desktop
C:\Users\[tu usuario]\Documents
C:\Users\[tu usuario]\Downloads
C:\Users\[tu usuario]\AppData (carpeta oculta)
Nota: Para ver carpetas ocultas, ve a Ver > Mostrar > Elementos ocultos en el Explorador de archivos.

Creacion de estructura de carpetas

Ahora crea la siguiente estructura de carpetas dentro de Documentos:

  1. Dentro de Documentos, crea una carpeta llamada Curso_IFCT0309.
  2. Dentro de Curso_IFCT0309, crea tres subcarpetas: Practicas, Apuntes y Capturas.
  3. Crea un archivo de texto dentro de Practicas llamado prueba.txt con el texto "Esto es una prueba de la Practica Windows".
  4. Copia el archivo prueba.txt a la carpeta Apuntes.
  5. Mueve el archivo original de Practicas a la carpeta Capturas.
  6. Comprueba que ahora prueba.txt esta en Apuntes (copia) y Capturas (movido).

Atajos de teclado utilizados

Accion realizada Atajo de teclado usado (si lo usaste)
Crear carpeta
Copiar archivo
Mover archivo (cortar + pegar)
Renombrar archivo/carpeta

Parte 3 - Administrador de tareas (30 min)

Procedimiento

  1. Abre el Administrador de tareas con Ctrl + Shift + Esc (o clic derecho en la barra de tareas).
  2. Si aparece en modo simplificado, haz clic en Mas detalles.
  3. Explora cada pestana y documenta lo que encuentras.

Pestanas del Administrador de tareas

Pestana Que informacion muestra Dato relevante que observas
Procesos
Rendimiento (CPU)
Rendimiento (Memoria)
Rendimiento (Disco)
Inicio (Aplicaciones de arranque)
Servicios
Nota: Fijate en el uso de CPU y RAM con el sistema en reposo. Esto te dara una idea de cuantos recursos consume Windows 10 sin aplicaciones abiertas.

Acciones con procesos

Realiza las siguientes acciones y documenta el resultado:

Accion Resultado observado

Observa cuanta RAM consume el Bloc de notas
Finaliza el proceso del Bloc de notas desde el Administrador
Abre 5 pestanas en el navegador Edge y observa el consumo de RAM
Cierra las pestanas y observa si la RAM se libera inmediatamente

Parte 4 - Linea de comandos: CMD y PowerShell (40 min)

Windows incluye dos interfaces de linea de comandos: CMD (Simbolo del sistema, clasico) y PowerShell (mas moderno y potente). Abre CMD buscando "cmd" en el menu de inicio.

Comandos en CMD

Comando Que hace Resultado (copia la salida)
hostname Muestra el nombre del equipo
ipconfig Muestra la configuracion de red
systeminfo | findstr /i "nombre sistema" Muestra info del sistema
dir C:\Users Lista el contenido de la carpeta Users
cd %USERPROFILE%\Documents Cambia al directorio Documentos
mkdir CMD_prueba Crea una carpeta llamada CMD_prueba
echo Hola desde CMD > saludo.txt Crea un archivo de texto
type saludo.txt Muestra el contenido del archivo
tree Curso_IFCT0309 Muestra arbol de carpetas

Comando PowerShell Equivalente en CMD Resultado

Parte 5 - Gestion de usuarios (30 min)

Procedimiento

  1. Ve a Configuracion > Cuentas > Otros usuarios (o Familia y otros usuarios).
  2. Haz clic en Agregar otra persona a este equipo.
  3. Selecciona No tengo los datos de inicio de sesion de esta persona.
  4. Selecciona Agregar un usuario sin cuenta de Microsoft.
  5. Crea un usuario local con los siguientes datos.

Datos del nuevo usuario

Dato Valor que has puesto
Nombre de usuario
Tipo de cuenta
Contrasena

Prueba del nuevo usuario

  1. Cierra sesion e inicia sesion con el nuevo usuario.
  2. Observa que el escritorio esta limpio (nuevo perfil).
  3. Vuelve a iniciar sesion con tu usuario original.

Que diferencias has observado al iniciar sesion con el nuevo usuario?

Comandos de gestion de usuarios (CMD como administrador)

Desde CMD (como administrador), ejecuta estos comandos de gestion de usuarios:

Comando Que hace Resultado

Parte 6 - Herramientas de mantenimiento (30 min)

Liberador de espacio en disco

  1. Busca "Liberador de espacio en disco" en el menu de inicio (o ejecuta cleanmgr).
  2. Selecciona la unidad C: y espera el analisis.
Tipo de archivo Espacio que ocupa Marcado para eliminar?
Archivos temporales de Internet
Archivos temporales de Windows
Papelera de reciclaje
Miniaturas
Espacio total que se puede liberar

Desfragmentacion y optimizacion

  1. Busca "Desfragmentar y optimizar unidades".
  2. Documenta el estado de la unidad:
Unidad Tipo de medio Estado Ultima optimizacion
C:
Nota: En un SSD, Windows no desfragmenta sino que ejecuta un comando TRIM para optimizar el rendimiento. La desfragmentacion clasica solo se aplica a discos duros mecanicos (HDD).

Programador de tareas

  1. Busca "Programador de tareas" en el menu de inicio.
  2. Crea una tarea basica:
    • Nombre: "Practica_tarea_programada"
    • Descripcion: "Tarea de prueba para la practica"
    • Desencadenador: Una vez
    • Accion: Iniciar un programa > notepad.exe
  3. Ejecuta la tarea manualmente (clic derecho > Ejecutar) y comprueba que se abre el Bloc de notas.

Parte 7 - Preguntas de reflexion

Responde a cada pregunta con un minimo de 3 lineas. Las respuestas deben reflejar tu experiencia personal durante la practica.

1. Que utilidad tiene conocer la estructura de directorios de Windows? En que situaciones laborales necesitarias navegar por carpetas del sistema como C:\Windows o C:\Program Files?

2. Que diferencias has notado entre usar el Explorador de archivos (interfaz grafica) y la linea de comandos (CMD) para gestionar archivos? Cual te parece mas eficiente y por que?

3. Por que es importante monitorizar los procesos y el rendimiento del sistema con el Administrador de tareas? Describe un escenario real en el que esta herramienta te seria util.
